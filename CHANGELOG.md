# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-04-05

### Added

- 55 tools across 10 categories
- Multi-agent system with specialized agent types
- Multi-LLM support via LiteLLM (Claude, GPT, Gemini, Ollama)
- Multi-PC deployment with worker fleet management
- 21-page React admin panel with dark mode
- Visual workflow builder
- 12 agent presets
- Agent-to-agent communication
- Knowledge base with categories and search
- Scheduled tasks with cron automation
- Real-time analytics
- Integrations: Slack, Jira, GitHub, Notion, Telegram, Google Workspace
- Role-based access control
- Audit logging
- Docker Compose deployment
- Prometheus + Grafana monitoring
- GitHub Actions CI/CD
